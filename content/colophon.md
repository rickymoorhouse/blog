## Technology

This site is built using [Hugo](https://gohugo.io) and custom scripts through Github Actions.  It is hosted on [BunnyCDN](https://bunny.net/). [More details on how the blog is built](/projects/blog).  For analytics I use [Umami](https://umami.is).

## Typography

The font families used are based on [Modern Font Stacks](https://modernfontstacks.com/) from which I use the *Industrial* stack for headings and *Transitional* for body content.For additional elements I use `system-ui`
